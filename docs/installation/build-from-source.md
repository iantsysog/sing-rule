# Build From Source

## :material-graph: Requirements

* Go 1.25.x

Install Go from [https://go.dev/doc/install](https://go.dev/doc/install). The latest stable version is recommended.

## :material-fast-forward: Build

```bash
make
```

Build and install the binary to `$GOBIN`:

```bash
make install
```
